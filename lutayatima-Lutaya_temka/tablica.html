h1 h2 ... h6 это размер текста

p новый абзац

b жирный текст

i текст курсивом

strike перечёрк

sub это типо маленькие над строкой как в формулах

sup это над строкой типо степень степь тайга тундра

br конец строки (всё что после на другой строке)

hr типо линия такая на всю строку

abbr + атрибут тип для аббривиатур ( если навестись то будет текст который ты напишешь)

addres для адресов, просто типо крусив

big текст становится больше

small текст становится меньше

pre текст с сохранением всех пробелов, отступов и т.д.

code тут пишут небольшой кусок кода

ol лист текста

start с какого числа начнётся счёт

type каким образом выглядит

a буквы

i римские

li элемент листа

ul лист с кругом( или тем что ты напиешь как в ol )

a href="..." переход на дргую страницу

a href="..." target куда перейти

_blank в новом окне, _self в этой вкладке

!!img картинки(закрывать не надо)!!

width широта картинки

height высота картинки

alt текст если картинка не прогрузилась

table border="..." таблица, числа меняют контуры

align="center" отцентрировать текст

link + rel="stylesheet" href="" продключить файл css(не требует закрытия(одиночный))

script src="" подключить файл javascript

div можно написать текст.Создаёт блок, позже через css можно добавить любой стиль.Занимает весь экран

span то же что и div, но занимает не всё пространство экрана.Можно комбинировать с div, чтобы стиль был только на опред текст, а не на весь

form action"" method внутри пишем формы для ввода и тд.(action передаёт данные на другую страницу, сервер и тд. Атрибутов много, эти два обязательны)

label="" подпись для поля

input type="text" само поля для ввода.Атрибутов много, type обязателен (не нуждается в закрытии)

placeholder="" атрибут для input, подсказка в поле ввода

value="" атрибут для input, то что будет изначально написанно

maxlength="" атрибут для input, максимум символов

readonly атрибут для input, только для чтения

name="" атрибут для input, функционально имя для поля

id="" атрибут для input, id обьекта

lable for атрибут для lable, при нажатии на надпись возле поля текст становится активным

input type="password" поле для ввода пароля, будет звёздочками

input type="email" тот же текст, но в разных браузерах может быть разное

input type="url" тот же текст, просто правильнее для url

input type="color" поле ля выбора цвета(всё то ниже для input)

date выбор даты

time выбор времени

checkbox ставить/убирать галочку в квабрате

radio ставить/убирать галочку в круге

file загрузить файл на сервер

buttom просто кнопка

submit кнопка, перезагружающая страницу

reset кнопка, очищающая все данные, введённые пользователем(дальше уже не для input)

textarea создаёт поле, которое может растягивать сам пользователь

textarea required делает форму обязательной

select выпадающий список

option элементы выпадающего списка

selected опция option, изначально будет выбран этот вариант

disabled оция option, изначально этот обьект нельзя выбрать

multiple атрибут для select, можно выбрать несколько вариантов

header шапка сайта

main основная часть сайта

aside боковая часть сайта

footer футер

progress прогрес бар

max атрибут progress, максимум прогрес бара

details детали

summary выдвижной текст
